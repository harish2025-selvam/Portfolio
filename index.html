About Me

Hello! I'm Harish S, a Computer Science student at VIT Vellore with a deep passion for Artificial Intelligence, Machine Learning, Computer Vision, and Full-Stack Development. I love building end-to-end solutions that bridge cutting-edge AI with real-world impact.

My project portfolio spans multiple domains: AI & Medical Imaging — a Breast Cancer Detection web app using a CNN + Transformer hybrid model; Computer Vision — an Automatic Number Plate Recognition (ANPR) system powered by YOLOv8 and EasyOCR with real-time alerts; Mobile AI — a Plant Disease Detector with a 75-class CNN model paired with a React Native app; Cloud & Serverless — a production-ready Mule-to-AWS Lambda migration framework with 40+ component mappings; Full-Stack SaaS — iFlux, a multi-tenant BPMN-style workflow engine built in Go, React, and PostgreSQL with Keycloak auth, Redis queuing, and Stripe billing; IoT & Automation — a virtual smart home simulator using event-driven architecture and MQTT; and NLP — a customer feedback sentiment analysis dashboard using TextBlob.

I hold certifications in Programming in C, Full Stack Development, Ethical Hacking, Data Analytics, Networking, and Generative AI. Beyond academics, I've won hackathons and volunteered in blood donation drives, tech education for school students, and disaster relief support. My goal is to become an impactful AI/ML developer who builds solutions that truly matter.
